# Cloud Web Application – AWS (Beginner Project)

## About the Project
This is a beginner cloud project created to understand how a web application is deployed on AWS.
The application is hosted on an EC2 instance.

## Cloud Services Used
- AWS EC2 – Virtual server
- AWS IAM – Secure access
- AWS Security Groups – Network security
- AWS CloudWatch – Monitoring

## Architecture
User → Internet → EC2 → Web Application

## How I Deployed
1. Created an EC2 instance (Amazon Linux)
2. Installed web server
3. Deployed application code
4. Configured security group for HTTP access
5. Verified application using public IP

## What I Learned
- Basics of cloud deployment
- Importance of security groups
- How monitoring works in AWS
